# 84th Putnam Competition (2023) Solutions

## About Me
I'm a person passionate about mathematics, especially competition math and rigorous proof techniques. This repository contains my solutions to the 84th Putnam Competition problems.

## Problems Covered
### A Problems
- A1: Calculus Computation
- A2: Polynomial Interpolation
- A3: Differential Inequality
- A4: Density of Icosahedron Combinations
- A5: Generating Function
- A6: Game Theory

### B Problems
- B1: Lattice Path Counting
- B2: Binary Weight Minimization
- B3: Expected Alternating Subsequence
- B4: ODE Optimization
- B5: Permutation Square Roots
- B6: Smith Matrix Determinant
